# 临时邮箱自动注册工具

一个用于自动生成临时邮箱、接收验证码并完成注册的工具。

## 功能特点

- 🎲 随机生成账号密码
- 📬 自动生成临时邮箱
- 📨 发送验证码
- 🔄 自动检测并提取邮件中的验证码
- ✅ 一键完成注册

## 部署说明

本项目已配置为可直接部署到 Vercel。

### 部署步骤

1. Fork 本仓库
2. 在 Vercel 中导入项目
3. 点击 "Deploy" 按钮
4. 部署完成后即可访问

## 本地开发

1. 克隆本仓库
2. 直接在浏览器中打开 `index.html` 文件
3. 开始使用工具

## 注意事项

- 本工具仅用于临时注册使用
- 生成的邮箱有效期有限，请及时使用
- 请遵守相关网站的使用条款

## 技术栈

- HTML5
- CSS3
- JavaScript
- Mail.tm API
